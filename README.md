# 🚀 Jenkins CI Pipeline

## 📌 Project Overview

This project demonstrates the implementation of **Continuous Integration (CI)** using **Jenkins**. The pipeline is integrated with GitHub and automates the process of retrieving source code, installing dependencies, running automated tests, and validating the build.

The goal of this project is to showcase how Jenkins can be used to automate software development workflows and improve code quality through Continuous Integration practices.

## ⚙️ Jenkins Pipeline Stages

### 1. Check Python

Verifies that Python is installed and accessible.

### 2. Install Dependencies

Installs the required project dependencies listed in `requirements.txt`.

### 3. Run Tests

Executes automated test cases using Pytest.

### 4. Build Success

Confirms that all pipeline stages have completed successfully.
